Perguntas frequentes

Respostas rápidas para as dúvidas mais comuns.

Quando o app será lançado?

Estamos em fase de desenvolvimento e testes. As datas serão comunicadas no Instagram.

O DailyMed vai ser gratuito?

O objetivo é disponibilizar uma versão base gratuita para utilizadores domésticos.

Como são tratados os meus dados?

Os dados são usados apenas para o funcionamento do app e não para fins comerciais.

Como registo os meus medicamentos?

O registo será feito manualmente, com campos simples para nome, dosagem e posologia.

Posso usar para familiares?

Sim. A ideia é permitir uma gestão simples para cuidadores e famílias.
