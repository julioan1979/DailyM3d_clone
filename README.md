# dailymed
Landing page do DailyMed, um projeto academico focado em melhorar a gestao domestica de medicamentos.

## objetivo
Divulgacao publica do app para utilizadores finais, com linguagem simples e foco em beneficios.

## publico
Utilizadores finais, cuidadores e pessoas com rotinas de medicacao complexas.

## estrutura
- Hero com proposta de valor e CTA.
- Secoes de problema, beneficios, funcionalidades e seguranca.
- Contexto do projeto e links de acompanhamento.
